This folder contains details of the nodes in the clusters generated by the infoMap community algorithm on the directed graph. 

Consider looking at imDComSize.txt to gain a brief overview of the stats

The files Imd[0-9]+WithDetails contain the product ids as well as the title and category in that particular cluster.

The detail files were generated from the files w/o detail (in the zipped file) by running the ClusterNodes.java file on the former.
